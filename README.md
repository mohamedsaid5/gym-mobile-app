# My Gym App

A mobile gym workout tracking application built with Kivy and Python. Track your workouts, manage your profile, and follow structured workout plans with integrated video tutorials.

**Created with AI**

## Features

- User profile management (name, age, weight, height)
- View workout plans with exercise video links
- Log workouts with sets, reps, and dates
- View workout history

## Installation

```bash
pip install kivy
python main.py
```

## Building for Android

```bash
pip install buildozer
buildozer android debug
```

## Technologies

- Python
- Kivy
- SQLite3
